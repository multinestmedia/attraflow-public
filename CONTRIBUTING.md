## Code Contributions

**Note:** AttraFlow uses a proprietary license. The source code is visible for educational purposes, but modifications and distribution are restricted.

### For Small Fixes

- Typos in documentation
- Minor bug fixes
- UI/UX improvements

Please open an issue first to discuss the change before submitting a pull request.

### For Major Changes

All significant code contributions require:

1. **Prior discussion** in Issues or Discussions
2. **Written approval** from maintainers
3. **Contributor License Agreement** (CLA) if required

## Community Guidelines

### Code of Conduct

- **Be respectful** and inclusive
- **Be constructive** in feedback
- **Respect privacy** - never share personal data or screenshots containing sensitive information
- **Stay on topic** - AttraFlow is about personal insight and self-discovery

### Response Times

- We're a small team and this is a passion project
- **Issues**: We aim to respond within 1 week
- **Feature Requests**: May take longer to evaluate
- **Security Issues**: Please email directly (see below)

## Security Issues

**Do not report security vulnerabilities in public issues.**

If you discover a security vulnerability, please email:
- **hello@multinestmedia.com**
- Or open a private security advisory on GitHub
